## 需求清单（绝对标准，不可偏离）

### 一、直播/视频商品浮层卡片（叠加在视频播放区域上）
卡片必须**同时包含以下 5 项**，缺一不可：
- 商品图（Image.network 或 Image.asset）
- 商品标题（Text）
- 商品价格（Text，如 "¥99.00"）
- 优惠信息（至少展示一种：满减 / 优惠券 / 直降；无优惠时必须显示 "暂无优惠" 占位）
- 库存状态（如 "现货" / "库存紧张" / "缺货"）

### 二、点击卡片 → 弹出商品详情半屏浮层
- 必须使用 `showModalBottomSheet` 或类似从底部上滑的半屏组件（`isScrollControlled: true`），**不能是全屏弹窗、不能是新页面跳转**。
- 半屏浮层内必须包含以下三个模块（缺一不可）：
  1. **规格**（如尺寸、颜色选择器，或明确的规格文字说明，不能为空）
  2. **优惠**（与卡片保持一致或更详细；无优惠则显示 "无优惠活动"）
  3. **评价摘要**（例如 "好评率 98%" 或 "共 342 条评价"，必须有具体摘要）

### 三、半屏浮层内的操作按钮
- **同时存在两个独立按钮**：  
  - `加入购物车`：点击后要有反馈（SnackBar / Toast / 购物车数量变化），加入购物车逻辑。  
  - `立即购买`：点击后跳转到下单确认页或拉起购买流程，**行为不能等同于加入购物车**。
- 两个按钮必须功能正常、互不干扰。
