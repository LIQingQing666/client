  Dart Frog 后端 — JSON 文件持久化（新增）

  文件：lib/data/database.dart + routes/_middleware.dart

  机制：
  1. 启动时加载：_loadAll() 从 data/*.json 读取数据，若无文件则使用种子数据
  2. 定时自动保存：每 3 秒 Timer.periodic 将内存数据写入 JSON 文件
  3. 写操作即时保存：中间件在每次 POST/PUT/DELETE 请求成功后调用 persist()

  持久化文件（data/ 目录下）：
  - users.json、videos.json、products.json
  - comments.json、orders.json、carts.json
  - video_likes.json、_meta.json（ID 计数器）

  生效方式

  两个后端代码均已通过编译检查（tsc --noEmit 零错误，dart analyze 零错误）。需要重启对应服务使改动生效：

  # Node.js 后端
  cd D:\lqq\client\apps\server
  npm run dev

  # Dart Frog 后端
  cd D:\lqq\client\后端服务-RESTful API
  dart_frog dev --port 8080

  Node.js 后端重启后种子数据不会重置已有数据。Dart Frog 后端重启后会自动从 data/ 目录加载之前持久化的 JSON 数据。
