### 二、视频后台播放问题

**问题**：切换到其他界面（直播、购物车、订单、我的）时，视频仍在播放

**修复要求**：

- 页面切换时，自动暂停当前视频
- 返回视频界面时，自动恢复播放
- 使用 `AutomaticKeepAliveClientMixin` 或监听页面可见性

**实现方案**：

在 `FeedPage` 中添加生命周期监听：

@override
void didChangeAppLifecycleState(AppLifecycleState state) {
  if (state == AppLifecycleState.paused) {
    // 页面不可见，暂停所有视频
    _pauseAllVideos();
  } else if (state == AppLifecycleState.resumed) {
    // 页面可见，恢复当前视频
    _resumeCurrentVideo();
  }
}

或者使用 `VisibilityDetector` 包裹页面。

**涉及文件**：`lib/pages/feed/feed_page.dart`
