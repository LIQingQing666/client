本项目实现了直播间实时通信功能，包含：
- **后端**：Node.js + Fastify 5 + Socket.IO（位于 `apps/server/`）
- **前端**：Flutter 移动端（位于 `apps/mobile/`）+ 浏览器测试客户端
- **核心能力**：WebSocket 实时评论、弹幕滚动、在线人数广播、讲解商品推送

## 二、实时功能架构

### 2.1 WebSocket 事件表

| 方向 | 事件名 | 说明 |
|------|--------|------|
| 客户端→服务端 | `join` | 加入直播间，参数 `{room: "房间ID"}` |
| 客户端→服务端 | `leave` | 离开直播间 |
| 客户端→服务端 | `send_message` | 发送评论，参数 `{room, user_id, content}` |
| 客户端→服务端 | `set_explaining_product` | 主播切换讲解商品，参数 `{room, product_id}` |
| 服务端→客户端 | `danmaku` | 收到弹幕/评论 |
| 服务端→客户端 | `online_count` | 在线人数变化，参数 `{count: N}` |
| 服务端→客户端 | `room_state` | 进入房间时的当前状态 |
| 服务端→客户端 | `explaining_product` | 讲解商品切换通知 |
| 服务端→客户端 | `gift_sent` | 收到礼物通知 |

### 场景 4：弹幕滚动动画

**目的**：验证弹幕覆盖在视频区域上方，从右向左滑动飘过。

**步骤**：

1. 发送多条消息（连续发送 5 条）
2. 观察视频区域上方的弹幕

**预期结果**：

- 每条弹幕都是一个半透明气泡（用户消息：黑底白字；系统消息：橙色底）
- 弹幕从右侧边缘出现，以不同速度匀速滑向左侧，移出屏幕后消失
- 多条弹幕出现在不同的纵向位置（随机分布在视频上半区域）
- 最多同时显示 12 条弹幕，超过后移除最早的

**技术实现**：Flutter 端使用 `AnimationController` + `TickerProviderStateMixin`，每条弹幕独立动画，时长 4~7 秒随机。

### 场景 5：讲解商品切换（HTTP 触发）

**目的**：验证通过 HTTP 接口切换讲解商品后，所有客户端实时更新。

**步骤**：

1. 打开两个浏览器标签页（客户端 A 和 B）
2. 获取一个商品 ID：

```cmd
curl "http://localhost:3000/api/products?page=1&page_size=1"
# 记下返回的 id 字段
```

3. 通过 curl 触发讲解商品切换：

```cmd
curl -X PUT "http://localhost:3000/api/live/rooms/<直播间ID>/explaining_product" \
  -H "Content-Type: application/json" \
  -d "{\"product_id\":\"<上一步的商品ID>\"}"
```

> 直播间 ID 可通过 `curl "http://localhost:3000/api/live/rooms"` 获取。

**预期结果**：

- 返回 `{"code":0,"data":{"room_id":"...","product_id":"...","product_name":"..."},"message":"讲解商品已切换"}`
- 两个浏览器标签页的顶部都弹出「主播正在讲解」商品卡片
- 卡片显示商品名称、价格、封面图
- Flutter 客户端同步显示讲解商品卡片

**实现原理**：

```
HTTP PUT → live.ts 路由 → getIO().to(roomId).emit('explaining_product', ...)
                                        ↓
                         所有 roomId 房间内的客户端收到事件
                                        ↓
                         Flutter: live_provider.dart 处理 eventName='explaining_product'
                         浏览器: socket.on('explaining_product', ...)
```

### 6.1 后端：加入房间时推送预置评论

`apps/server/src/websocket/live.ts`:

```typescript
socket.on('join', (data) => {
  const room = data.room ?? data.namespace ?? 'default';
  socket.join(room);

  // 更新在线人数
  state.onlineCount += 1;
  io.to(room).emit('online_count', { count: state.onlineCount });

  // 发送 4 条预置评论（延迟递进）
  const historyComments = [
    { user_name: '小明数码', content: '这款真的超好用，我已经回购三次了！', delay: 300 },
    { user_name: '小红穿搭', content: '主播今天讲解的好详细啊，学到了', delay: 600 },
    { user_name: '阿杰户外', content: '已下单，期待收货！', delay: 900 },
    { user_name: '系统消息', content: '欢迎进入直播间！', delay: 1200, type: 'system' },
  ];
  historyComments.forEach((c) => {
    setTimeout(() => {
      socket.emit('danmaku', {
        id: crypto.randomUUID(),
        user_name: c.user_name,
        content: c.content,
        type: c.type ?? 'user',
      });
    }, c.delay);
  });
});
```

### 6.2 后端：讲解商品 HTTP 端点

`apps/server/src/routes/live.ts`:

```typescript
app.put('/api/live/rooms/:roomId/explaining_product', async (req, res) => {
  const { roomId } = req.params;
  const { product_id } = req.body;

  const product = db.prepare(
    'SELECT id, name, cover_url, price, original_price, sales, ai_sales_point FROM products WHERE id = ?'
  ).get(product_id);

  if (!product) {
    return res.status(404).send({ code: 404, message: '商品不存在' });
  }

  getIO().to(roomId).emit('explaining_product', {
    product,
    timestamp: new Date().toISOString(),
  });

  return { code: 0, data: { room_id: roomId, product_id, product_name: product.name }, message: '讲解商品已切换' };
});
```
